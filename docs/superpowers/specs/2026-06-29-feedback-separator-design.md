# Дизайн: разделитель между фидбэком и новым словом

Дата: 2026-06-29.

## Проблема
После ответа сообщение = фидбэк по прошлому слову (`✅/❌/💡` + формы + перевод)
сразу за ним новое слово. Оба блока похожи («to X - …»), граница неясна.

## Решение
В рендере экрана `quiz` (Тест и Учить), когда есть `Feedback`, между ним и
заданием нового слова вставляем **линию-разделитель + метку «🆕 Новое слово»**:

```
✅ Верно!
to go - went - gone - идти
➖➖➖➖➖➖➖➖
🆕 Новое слово
to forbid
Введите 3 формы по порядку:
```

Если `Feedback` пустой (первое слово сессии) — разделитель не добавляется.
Экран результата Теста (`test_result`) не меняется (там нет нового слова).

## Файлы
- `internal/bot/screens.go`: хелпер `withNewWord(feedback, prompt)` + константа
  разделителя; применить в ветках `ScreenQuiz` (learn и test).
- Тесты `internal/bot`: с фидбэком — есть линия и «🆕 Новое слово» перед
  заданием; без фидбэка — нет.

## Вне объёма
Логика квиза; экран результата.
