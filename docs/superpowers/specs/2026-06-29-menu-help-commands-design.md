# Дизайн: команды /menu и /help

Дата: 2026-06-29.

## Команды (текстовые)
Роутер перехватывает их в `Handle` до обработки как ответа в квизе:
- **/menu** → `svc.OpenMenu` → главное меню (очищает квиз/список-сессию, как «⬅️ Меню»).
- **/help** → статическое сообщение со ссылкой на гайд:
  ```
  📖 Как пользоваться ботом:
  https://github.com/irbgeo/go-irregular-verbs-tgbot/blob/main/docs/USER_GUIDE.md
  ```
  (Репозиторий приватный — ссылка работает у тех, у кого есть доступ; по
  согласованию оставляем blob-ссылку.)

Список команд в BotFather (автодополнение) настраивается отдельно в UI; бот
обрабатывает сами команды — это и реализуем.

## Файлы
- `internal/bot/router.go`: константы `guideURL`/`helpText`; в `Handle` ветки
  `/menu`→`handleMenu`, `/help`→`handleHelp`.
- Тесты `internal/bot/router_test.go`: `/menu` → экран меню; `/help` → сообщение
  со ссылкой.

## Вне объёма
Регистрация команд через setMyCommands (делается в BotFather UI); локализация.
