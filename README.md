# irregular-verbs-tgbot

## 🚀 Первый запуск

Пользователь пишет `/start`

Бот предлагает:
1. **Выбрать уровень английского** (elementary / pre-int / int / upper-int / advanced / proficiency)
   - Это будет начальный пул слов, которые ему покажут
2. **Выбрать формы** (британский или американский вариант)
3. **Выбрать порядок изучения** (по алфавиту или рандомно)

Готово — пользователь переходит в основное меню

---

## 📚 Основное меню (4 кнопки)

1. **Учебный материал** — просмотр и отбор слов
2. **Начать обучение** — если есть отобранные слова
3. **Мои слова** — посмотреть какие слова учит и какие пропустил
4. **Список слов** — все слова по уровням с метками (учу / пропустил)

---

## 📖 Режим "Учебный материал"

Бот показывает слово по одному:

```
🔤 TAKE (took, taken)
📝 Брать, взять

[✅ Хочу учить]  [❌ Не хочу учить]
```

**Логика:**
- Если нажал "Хочу учить" → слово добавляется в персональный список для обучения
- Если нажал "Не хочу учить" → слово исчезает из этого режима (видно будет только в списке всех слов)
- После каждых 5 добавленных слов → предложение: "Начать обучение?" 
- Пользователь может продолжить просмотр или перейти к обучению

---

## 🎓 Режим "Изучение-1" (множественный выбор)

**Цель:** 5 правильных ответов подряд для каждого слова

Бот показывает либо форму, либо перевод:

```
Выбери первую форму:
GET (past - ?, past participle - ?)

[GET]  [GOT]  [GAT]  [GED]  [GETTING]
```

Пользователь нажимает одно слово → получает результат (✅ или ❌)

Затем показывается вторая форма (прошедшее время):

```
Выбери past tense:
GET (got, gotten)

[GET]  [GOT]  [GETTING]  [TAKE]  [TOOK]
```

И третья форма (причастие):

```
Выбери past participle:
GET (got, gotten)

[GET]  [GOT]  [GOTTEN]  [TAKEN]  [GIVEN]
```

Потом перевод:

```
Что значит: GET?

[Идти]  [Брать]  [Давать]  [Говорить]  [Делать]
```

**Когда слово "сдано" из Режима-1:**
- Нужно 5 правильных ответов подряд
- После этого слово переходит в **Режим-2**
- В Режиме-1 больше не появляется

---

## ✍️ Режим "Изучение-2" (свободный ввод)

**Цель:** 5 правильных ответов подряд для каждого слова

Бот показывает форму или перевод и ждёт письменного ответа:

```
Напиши past tense глагола TAKE:
_________________
```

Пользователь пишет: `took`

Бот проверяет (учитывает опечатки, разные вариантные формы):
- ✅ Правильно!
- ❌ Неправильно! Правильный ответ: took

Или:

```
Напиши перевод глагола GO:
_________________
```

Пользователь пишет: `идти` или `ходить` или `пойти`

Бот проверяет против базы синонимов.

**Когда слово "полностью выучено":**
- 5 правильных ответов подряд в Режиме-2
- Слово отмечается как ✅ выученное
- Больше в обучении не появляется

---

## 📋 Мои слова

Пользователь видит:
- Слова, которые он сейчас учит (с прогрессом: в Режиме-1 или Режиме-2)
- Примерно: "GET — Режим 2 (3/5 правильных)"

---

## 📚 Список слов по уровням

Пользователь может посмотреть все слова в базе:

```
Elementary:
TAKE (учу)
GO (пропустил)
GET (выучено ✅)

Pre-intermediate:
...
```

Метки показывают статус для каждого слова.

---

## 🔔 Ежедневные задания

Если пользователь **24 часа не учился**:
- Бот автоматически отправляет задание (одно случайное слово из его текущих)
- Кнопка "Начать обучение"

---

## 🔄 Добавление новых слов для обучения

Когда пользователь выучил много слов из текущего уровня:
- Бот предлагает: "Хочешь добавить слова из следующего уровня?"
- Пользователь выбирает → переходит в Учебный материал для нового уровня

---

## 📊 Полный цикл для одного слова

1. **Учебный материал** → пользователь видит слово, нажимает "Хочу учить"
2. **Режим-1** → выбирает из 5 вариантов 4 раза (3 формы + перевод), нужно 5 правильных подряд
3. **Режим-2** → пишет ответ 5 раз подряд (5 правильных)
4. **Статус:** ✅ Выучено

Слово больше не появляется в обучении.
