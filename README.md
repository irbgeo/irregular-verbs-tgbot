# irregular-verbs-tgbot

Telegram-бот для заучивания английских неправильных глаголов. Две активности:
**Тест** по уровню (быстро находим слова, которые не знаешь) и **Учить**
(интервальная тренировка отобранных слов).

## 🚀 Старт

При первом запуске (`/start`) бот спрашивает **вариант форм**: британский
(`gb`) или американский (`us`). Это всё — дальше главное меню.

## 📋 Главное меню (3 кнопки)

1. **🧪 Тест** — выбрать уровень и пройти тест по его словам.
2. **🎓 Учить** — тренировать отобранные слова (интервально).
3. **📋 Мои слова** — посмотреть свои слова с фильтром.

## 🧪 Тест

Выбираешь уровень (Elementary … Proficiency). Слова уровня идут **по одному в
случайном порядке**. На каждое слово — круг из 4 заданий со **свободным вводом**
(ответ печатаешь текстом):

1. по переводу — ввести инфинитив (base);
2. по base — ввести past;
3. по base — ввести past participle;
4. по base — ввести перевод.

Под заданием — кнопки **«💡 Помощь»** и **«⏭️ Скип»**.

- **Помощь** → бот показывает 3 формы и перевод; слово добавляется в изучение.
- **Ошибка** (в любом из заданий) → бот показывает правильный ответ; слово
  добавляется в изучение.
- **Всё верно без помощи** → бот спрашивает: **«В изучение»** или **«Скип»**.

Так тест наполняет твой список слов для изучения тем, что ты ещё не знаешь.

## 🎓 Учить

Тренируем слова, отобранные в изучение. Каждое слово проходит **две ступени**
по очереди — сначала режим 1, потом режим 2. На каждой ступени работает своя
«коробка» Лейтнера 0–5.

**Ступень 1 — «Выбрать подходящее» (множественный выбор).**
Показывается случайная форма слова; нужно по очереди выбрать **base · past ·
past participle · перевод** из вариантов:
- для форм — 4 варианта: правильная + 3 ошибочных (из частых ошибок слова и
  форм других глаголов);
- для перевода — 5 вариантов: правильный + 4 случайных перевода других слов.

**Ступень 2 — «Ввести подходящее» (свободный ввод).**
Показывается случайная форма слова; нужно ввести все 4 формы (base, past,
participle, перевод) текстом.

**Как двигается «коробка» (на каждой ступени):**
- весь круг верный → коробка **+1**;
- ошибка → коробка сбрасывается в **0**;
- **коробка 5 в режиме 1** → слово переходит в **режим 2** (коробка 0);
- **коробка 5 в режиме 2** → слово **выучено**.

**Подмешивание и веса.** Выученные слова не исчезают — иногда подкидываются для
повторения (свободным вводом). Выбор следующего слова — взвешенный: **90%**
слова в изучении, **10%** выученные. Одно и то же слово не повторяется чаще, чем
раз в **5** слов. Если на повторении выученное слово ошиблись — оно
возвращается в изучение (режим 2, коробка 0).

Когда **все слова выучены** (учить нечего) — бот предлагает **пройти тест**,
чтобы набрать новые слова.

## 📋 Мои слова

Список твоих слов с переключаемым фильтром:

- **в изучении** (с прогрессом — номер коробки);
- **выучено**;
- **скип**.

## 🏷️ Статусы слова

| Статус | Значение |
|--------|----------|
| новое | ещё не встречалось (нет в твоём списке) |
| в изучении | тренируется в «Учить»: режим 1 или 2, коробка 0–5 |
| выучено | прошёл режим 2 (коробка 5 в режиме 2) |
| скип | отмечено «не учить» |

## 📊 Полный цикл слова

1. **Тест** → ошибся / нажал «Помощь» → слово в **изучение** (режим 1, коробка 0).
   (Если ответил верно — выбираешь «в изучение» или «скип».)
2. **Учить, режим 1** (выбор) → коробка 5 → переход в **режим 2**.
3. **Учить, режим 2** (ввод) → коробка 5 → **выучено**.
4. Выученное иногда подкидывается для повторения; ошибка возвращает в изучение
   (режим 2, коробка 0).
